# Pi.Alert Context

## Basic Information

- **Name**: Pi.Alert
- **Slug**: pialert
- **Categories**: 4

## Description

Pi.Alert is a WIFI / LAN intruder detector. Checks the devices connected and alert you with unknown devices. It also warns of the disconnection of "always connected" devices.
