# MediaMTX Context

## Basic Information

- **Name**: MediaMTX
- **Slug**: mediamtx

## Description

MediaMTX is a ready-to-use SRT / WebRTC / RTSP / RTMP / LL-HLS media server and media proxy that allows you to read, publish, proxy, record and playback video and audio streams.
